# Cloudfin UI — 桌面端（Windows / Linux / macOS）

> 版本：v0.1.0  
> 日期：2026-04-03  
> 状态：**草案 — 待审查**

---

## 1. 概述

桌面端 UI 是独立于 Core 的原生应用，通过 HTTP API 与 Cloudfin Core 守护进程通信。面向**多任务、高密度信息**场景，强调**效率和专业感**。

**技术栈**：
- Windows：C# + WinUI 3 或 C++ + Qt
- macOS：Swift + SwiftUI
- Linux：Rust (QML) / GTK / Qt

**通信架构**：
```
┌──────────────────────────────────┐
│        Cloudfin UI (桌面应用)       │
│  (WinUI3 / SwiftUI / Qt / GTK)   │
└──────────────┬─────────────────────┘
               │ HTTP (127.0.0.1:19001)
               ▼
┌──────────────────────────────────┐
│        Cloudfin Core (守护进程)      │
│          127.0.0.1:19001          │
└──────────────┬─────────────────────┘
               │ .so / .dylib / .dll
               ▼
┌──────────────────────────────────┐
│           cloudfin-mod/            │
│    p2p / crdt / tor / i2p / storage│
└──────────────────────────────────┘
```

## 2. 窗口布局

### 2.1 主窗口结构

**单窗口 + 侧边栏布局**，充分利用大屏空间：

```
┌─────────────────────────────────────────────────────────────┐
│ [≡]  ☁ 云鳞 Cloudfin          [搜索...]   [_][□][×]      │  ← 标题栏
├──────────┬────────────────────────────────────────────────┤
│          │                                                │
│ 状态     │                                                │
│ 模块     │          主内容区                               │
│ 网络     │      （根据左侧选中项切换）                      │
│ 同步     │                                                │
│ 设置     │                                                │
│          │                                                │
├──────────┼────────────────────────────────────────────────┤
│          │ Core: ● 运行中   P2P: ● 3节点   同步: ● 2文档  │  ← 底部状态栏
└──────────┴────────────────────────────────────────────────┘
   220px                    剩余空间
```

**侧边栏**：固定宽度 220px，图标 + 文字标签，垂直排列

**主内容区**：根据选中项显示对应页面，支持多标签页

**状态栏**：固定在底部，显示关键指标实时聚合

## 6.2 图片背景图层架构

**核心原则**：背景层与内容层完全分离，背景固定不动，内容层覆盖其上。

**完整图层栈（底→顶）**：

```
┌──────────────────────────────────────────────────────┐  ← 图层0：根容器（Stack/Overlay）
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │ 图层1: Image.asset(图片)                    │  │  ← 充满整个窗口，无外边距
│  │ Window.SetBackgroundImage() 或 Stack 内 Image  │  │  ← 固定不滚动
│  └──────────────────────────────────────────────┘  │
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │ 图层2: 遮罩层                               │  │  ← Container(color)
│  │ Color.fromRGBO(0,0,0,遮罩强度)             │  │    alpha = 遮罩强度 × 255
│  └──────────────────────────────────────────────┘  │    范围: 0.40 ~ 0.70
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │ 图层3: 内容层                                │  │  ← 主应用内容区域
│  │ ┌──────────────────────────────────────────┐ │  │
│  │ │ 侧边栏  │  主内容区（固定背景）          │ │  │
│  │ │ 透明背景│  透明背景，内容区无独立背景    │ │  │
│  │ └──────────────────────────────────────────┘ │  │
│  │ 内容区背景: transparent                       │  │
│  │ 侧边栏背景: Color.fromRGBO(0,0,0,0.60)      │  │
│  │ 卡片背景: transparent                        │  │
│  │ 卡片边框: BorderSide(color:white.alpha(10))  │  │
│  └──────────────────────────────────────────────┘  │
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │ 图层4: 状态栏                                │  │  ← 底部固定状态栏
│  │ Container(color: Color.fromRGBO(0,0,0,0.70)) │  │  ← 自身保证可读性
│  └──────────────────────────────────────────────┘  │
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │ 图层5: Dialog / 悬浮面板                     │  │  ← 自带深色半透明背景
│  │ Container(color: Color.fromRGBO(30,30,30,0.95))│ │  ← 禁止继承透明父背景
│  └──────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
```

**遮罩强度预设**：
| 强度值 | 适用场景 |
|--------|---------|
| 0.40 | 极浅/模糊壁纸 |
| 0.50 | 默认推荐 |
| 0.60 | 中等复杂度壁纸 |
| 0.70 | 高饱和/复杂壁纸 |

### 6.3 图片背景组件规范

**桌面端窗口背景实现**：
```dart
// WinUI3 / WPF
window.SetBackground(new ImageBrush { ImageSource = wallpaper, Stretch = Stretch.UniformToFill });

// Qt / QML
ApplicationWindow {
  background: Image {
    source: wallpaperPath
    fillMode: Image.PreserveAspectCrop
    anchors.fill: parent
  }
}

// SwiftUI
.background(
  Image(wallpaperName)
    .resizable()
    .aspectRatio(contentMode: .fill)
    .frame(maxWidth: .infinity, maxHeight: .infinity)
    .clipped()
)
```

**侧边栏特殊处理**：
- 侧边栏背景: `Color.fromRGBO(0, 0, 0, 0.55~0.65)` 固定深色
- 不依赖透明度继承，保证在任何壁纸上可读
- 与主内容区分隔明显

**窗口尺寸变化**：
```
✓ 图片使用 PreserveAspectCrop（可能裁剪）
✓ 禁止使用 Stretch = Fill / None
✓ 图片位置: 居中 (Alignment.Center)
✓ 窗口resize时背景图片不动（fixed attachment）
```

**对话框/弹窗**：
```
✓ 弹窗自带背景: Color.fromRGBO(30, 30, 30, 0.96)
✓ 禁止直接继承透明父容器背景
✓ 可选叠加毛玻璃: blur sigma=8
✓ 弹窗边框: BorderSide(color: white.alpha(15))
```

**多显示器支持**：
```
✓ 仅主窗口显示壁纸
✓ 多窗口场景：子窗口使用纯色背景
✓ 托盘菜单: 纯色背景，不继承壁纸
```

### 6.4 主题色板

**暗色模式（默认）**：
- 背景：`#1E1E1E`
- 侧边栏：`Color.fromRGBO(37, 37, 37, 1.0)`（不透明深灰）
- 卡片背景：`transparent`（露出壁纸）
- 卡片边框：`BorderSide(color: white.alpha(0.08))`
- 主色：`#6C63FF`（紫蓝）
- 文字主：`#E0E0E0`
- 文字次：`#A0A0A0`
- 边框：`rgba(255,255,255,0.08)`
