# Cloudfin UI — 移动端（Android / iOS）

> 版本：v0.1.0  
> 日期：2026-04-03  
> 状态：**草案 — 待审查**

---

## 1. 概述

移动端 UI 是独立于 Core 的原生应用，通过 HTTP API 与 Cloudfin Core 守护进程通信。

**技术栈**：
- Android：Kotlin + Jetpack Compose
- iOS：Swift + SwiftUI

**通信架构**：
```
┌──────────────────┐
│  Cloudfin UI      │  ← 手机/平板
│  (Compose/SwiftUI)│
└────────┬─────────┘
         │ HTTP (127.0.0.1:19001)
         ▼
┌──────────────────┐
│  Cloudfin Core    │  ← 后台进程
│  (守护进程)       │
└────────┬─────────┘
         │ .so 插件
         ▼
┌──────────────────┐
│  cloudfin-mod/   │
│  p2p / crdt / ...│
└──────────────────┘
```

---

## 2. 导航结构

### 2.1 底部导航栏（5 Tab）

```
┌────────────────────────────────────┐
│ [状态]  [模块]  [网络]  [同步]  [设置] │
└────────────────────────────────────┘
```

每个 Tab 对应一个独立页面，支持左右滑动切换。

## 3. 页面详情

### 3.1 状态页（Status）

**路径**：`/` 或 `/status`

**功能**：显示 Core 和各模块运行状态

**布局**（单列卡片）：
```
┌────────────────────────────────┐
│ ☁ 云鳞                    v0.1.0│  ← 顶部栏
├────────────────────────────────┤
│ ┌────────────────────────────┐ │
│ │ ● 运行中                   │ │  ← 状态卡片
│ │ 已运行：2小时 34分          │ │
│ │ 设备ID：abcd-1234          │ │
│ └────────────────────────────┘ │
│ ┌────────────────────────────┐ │
│ │ 📡 P2P           ● 已连接  │ │  ← 模块状态列表
│ │ 🔒 TOR            ○ 已停止  │ │
│ │ 🌐 I2P           ● 已连接   │ │
│ │ 🔄 同步          ● 运行中   │ │
│ └────────────────────────────┘ │
│ ┌────────────────────────────┐ │
│ │ 流量：↑ 2.3 MB  ↓ 5.1 MB  │ │  ← 流量统计
│ └────────────────────────────┘ │
└────────────────────────────────┘
```

**API 调用**：
- `GET /api/core/status` → Core 状态
- `GET /api/modules` → 模块列表

**交互**：
- 下拉刷新
- 点击模块卡片 → 跳转 `/modules/{id}`

---

### 3.2 模块页（Modules）

**路径**：`/modules`

**功能**：管理 P2P / TOR / I2P / CRDT / Storage 模块

**布局**（垂直滚动卡片列表）：
```
┌────────────────────────────────┐
│ 模块管理                        │
├────────────────────────────────┤
│ ┌────────────────────────────┐ │
│ │ 📡 P2P 网络                 │ │
│ │ libp2p v0.54               │ │
│ │ ● 已连接  3 个节点          │ │
│ │                            │ │
│ │ [启动] [配置] [详情]       │ │
│ └────────────────────────────┘ │
│                                │
│ ┌────────────────────────────┐ │
│ │ 🔒 TOR 匿名网络            │ │
│ │ ○ 已停止                   │ │
│ │ [启动] [配置] [详情]       │ │
│ └────────────────────────────┘ │
│                                │
│ ┌────────────────────────────┐ │
│ │ 🔄 CRDT 同步               │ │
│ │ ● 运行中  2 个文档         │ │
│ │ [停止] [配置] [详情]        │ │
│ └────────────────────────────┘ │
└────────────────────────────────┘
```

**API 调用**：
- `GET /api/modules` → 模块列表
- `POST /api/modules` → 注册模块
- `DELETE /api/modules/:id` → 注销模块
- `POST /api/plugins/load` → 动态加载
- `DELETE /api/plugins/unload/:id` → 动态卸载

**交互**：
- 卡片点击 → 展开/收起详情
- [启动/停止] 按钮
- [配置] → 全屏配置页
- [详情] → 详情页

### 3.3 网络页（Network）

**路径**：`/network`

**功能**：P2P 网络可视化、节点管理

**布局**：
```
┌────────────────────────────────┐
│ P2P 网络                       │
├────────────────────────────────┤
│ ┌────────────────────────────┐ │
│ │    [节点A] ──── [本机]    │ │  ← 拓扑图
│ │       │           │       │ │
│ │   [节点B] ──── [节点C]   │ │
│ └────────────────────────────┘ │
│                                │
│ ● 已连接 3 个节点              │
│                                │
│ ┌────────────────────────────┐ │
│ │ 📍 节点A                   │ │  ← 节点列表
│ │ /ip4/192.168.1.x/tcp/... │ │
│ │ [断开] [详情]              │ │
│ ├────────────────────────────┤ │
│ │ 📍 节点B                   │ │
│ │ /ip4/192.168.1.y/tcp/... │ │
│ │ [断开] [详情]              │ │
│ └────────────────────────────┘ │
│                                │
│ [+ 添加节点]      [🔄 刷新]   │
└────────────────────────────────┘
```

**交互**：
- [添加节点] → 底部弹窗输入 Multiaddr
- 节点卡片 → 展开详情 / 断开连接

## 7. 状态管理（BLoC）

```
UI Layer (Compose/SwiftUI)
        ↓↑
   BLoC Layer
        ↓↑
   Repository
        ↓↑
   Core HTTP API
```

**BLoC 列表**：
- `CoreStatusBloc` → Core 状态
- `ModulesBloc` → 模块列表和操作
- `NetworkBloc` → P2P 网络状态
- `SyncBloc` → CRDT 同步状态
- `SettingsBloc` → 配置管理

---

## 8. 错误处理

| 情况 | UI 行为 |
|------|---------|
| Core 未启动 | 全屏弹窗："Core 未运行" + [启动 Core] |
| API 超时（5s） | Toast："连接超时" |
| API 错误 | Toast：显示错误信息 |
| 网络断开 | 顶部横幅："离线模式" |
